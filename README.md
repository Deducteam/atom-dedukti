# dedukti-editor

## Installation

 - open a terminal
 - go to the folder where you have installed the package
 - run the command make
 - install lptop.native (or a symlink of it) in the folder resource.

## Utilisation

| Action | Command |
|--|--|
| Activation | ctrl+alt+o |
| Check  | ctrl+alt+c |
| Erase color | ctrl+alt+e |
